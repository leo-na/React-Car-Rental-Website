
# 🚗 React Car Rental Website

Une plateforme moderne de location de voitures construite avec **React.js**. Ce projet permet aux utilisateurs de parcourir une flotte de véhicules, de consulter les détails techniques et de simuler une réservation via une interface fluide et responsive.

## 🚀 Fonctionnalités

* **Catalogue Dynamique :** Affichage des véhicules via une API JSON.
* **Navigation Fluide :** Utilisation de `react-router-dom` pour une expérience Single Page Application (SPA).
* **Système de Location :** Gestion de la sélection et du formulaire de réservation.
* **Design Responsive :** Interface optimisée pour mobile, tablette et desktop grâce à **Bootstrap**.
* **Composants Fonctionnels :** Utilisation intensive des **Hooks React** (`useState`, `useEffect`) pour la gestion d'état.

## 🛠️ Technologies utilisées

* **Frontend :** React.js (Hooks, Functional Components)
* **Styling :** Bootstrap 5 & CSS3
* **Routing :** React Router v6
* **Data :** API JSON / Fetch API
* **Icons :** Remix Icon / Font Awesome
